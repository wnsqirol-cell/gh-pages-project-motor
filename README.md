## 📋 Loyiha haqida

**Motorsikle Prime** — bu zamonaviy dizaynda qurilgan premium mototsikl savdo va ma'lumot saytidir. Sayt foydalanuvchilarga eng yangi va mashhur mototsikl modellarini ko'rish, mahsulotlar bilan tanishish va xarid qilish imkoniyatini beradi.

## 🚀 Loyihani lokal ishga tushirish

```bash
# 1. Repozitoriyani klonlash
git clone https://github.com/your-username/motorsikle-prime.git

# 2. Papkaga kirish
cd motorsikle-prime

# 3. index.html ni brauzerda ochish
open index.html
```

> Hech qanday qo'shimcha paket yoki o'rnatish talab etilmaydi — toza HTML/CSS/JS loyihasi.
